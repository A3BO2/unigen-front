# Unigen - 시니어를 위한 SNS AI 도움 어플

시니어 분들을 위한 간소화된 SNS 플랫폼입니다. 일반 인스타그램 스타일 모드와 시니어를 위한 간단한 Threads 스타일 모드를 제공합니다.

## 주요 기능

### 일반 모드 (인스타그램 스타일)
- 홈 피드 (스토리, 게시물)
- 검색 기능
- 게시물 업로드
- 알림
- 프로필 관리

### 시니어 모드 (Threads 스타일)
- **큰 글씨와 간단한 UI**: 시니어 친화적 디자인
- **대화형 업로드 모드**: 음성으로 글 작성 가능 (Web Speech API 사용)
- **AI 테마 말투 변환**:
  - 온화한 말투
  - 귀여운 말투
  - 손주에게 편지
  - 친구에게 안부
- **간소화된 네비게이션**: 홈, 글쓰기, 내 글 (3개만)

## 기술 스택

- React 18
- Vite
- React Router DOM
- Styled Components
- Lucide React (아이콘)

## 설치 및 실행

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 빌드
npm run build
```

## 프로젝트 구조

```
src/
├── components/
│   ├── normal/          # 일반 모드 컴포넌트
│   └── senior/          # 시니어 모드 컴포넌트
├── pages/
│   ├── onboarding/      # 온보딩/로그인 페이지
│   ├── normal/          # 일반 모드 페이지
│   └── senior/          # 시니어 모드 페이지
├── context/             # Context API (앱 모드 관리)
├── styles/              # 글로벌 스타일
└── App.jsx              # 메인 앱 및 라우팅
```

## 사용 방법

1. 웰컴 화면에서 모드 선택
   - "간단하게 시작하기" → 시니어 모드
   - "일반 방식으로 시작하기" → 일반 모드

2. 시니어 모드 로그인
   - 전화번호만 입력
   - 인증번호 확인
   - 간단하게 시작

3. 시니어 모드 글쓰기
   - "말로 쓰기" 버튼으로 음성 입력
   - 사진 찍기로 사진 추가
   - AI 테마로 말투 변환
   - 한 번에 업로드

## 주의사항

- 백엔드는 구현되지 않았으며, Mock 데이터를 사용합니다
- 음성 인식은 Web Speech API를 사용하며, Chrome 브라우저에서 가장 잘 작동합니다
- AI 말투 변환은 프론트엔드에서 Mock으로 처리됩니다 (실제로는 백엔드 API 필요)
